# ymark.wang

## Installation

```
git clone https://github.com/ymma/ymark.wang.git
```

## Install dependencies

```
npm install --registry=https://registry.npm.taobao.org --verbose
```

## Compile

```
npm run compile
```

## Start server

```
npm start
```

## How to visit

Open url `http://127.0.0.1:8360/` in browser.
